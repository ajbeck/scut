# Security Policy

## Supported Versions

Security fixes are provided for the latest released version of `scut`.

## Reporting a Vulnerability

Please report security issues privately by opening a GitHub security advisory for this repository.

Do not include sensitive exploit details in public issues. Include the affected version, a concise description, and reproduction steps when possible.

## Dependency Security

This project uses Dependabot for Go module and GitHub Actions updates. Security-related dependency updates should be reviewed and merged promptly after CI passes.
